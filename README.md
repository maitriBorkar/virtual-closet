# virtual-closet
A Django-based web application that helps users organize and manage their wardrobe digitally.

## Features

* User signup and login
* Upload wardrobe items
* Wardrobe view
* Layering view
* View closet inventory
* Weather widget

## Technologies Used

* Django
* Python
* HTML
* CSS
* SQLite

## Project Background

This project was developed during my internship as a virtual wardrobe management system.

## Future Improvements

* Outfit recommendations
* Clothing search and filters
* Mobile responsiveness improvements
* Cloud deployment
* Delete and edit items
  
## Setup

To enable the weather widget:

1. Create an account with OpenWeather.
2. Generate an API key.
3. Replace:

javascript
const apiKey = "YOUR_API_KEY";

with your own API key.
